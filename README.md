# 🎯 Number Guessing Game

A simple Python game where the player tries to guess a randomly generated number. Choose your difficulty and see if you can guess it in the fewest attempts!  

---

## Features

- Two difficulty levels:  
  - **Easy:** 1–50  
  - **Hard:** 1–100  
- Gives hints if the guess is too high or too low  
- Keeps track of your score  
- Runs in terminal/command prompt  
- Simple Python script — no extra packages needed  

---

## How to Run

1. Make sure **Python 3** is installed.  
2. Open terminal in the project folder.  
3. Run the game:
4. Follow the on-screen instructions:
Select difficulty level:
Plain text
1. EASY (1-50)
2. HARD (1-100)
Enter your guess and get hints until you find the number.
